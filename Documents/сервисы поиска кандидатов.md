# Сервисы поиска кандидатов

Ниже перечислены площадки, где в России соискатели обычно ведут резюме и откликаются на вакансии. Оценка в таблице относится к **возможности программного доступа** (API, открытые данные) для поиска и интеграции со стороны работодателя: наличие популярного сайта у пользователей не означает наличие публичного интерфейса для выгрузки кандидатов.

Источники по отдельным сервисам: официальные страницы API и открытых данных (HeadHunter, SuperJob, Avito, Трудвсем, документация Microsoft по LinkedIn), сверка статуса VK Работа на [rabota.vk.com](https://rabota.vk.com/) (сервис закрыт). Сведения о тарифах и полноте выдачи по API нужно уточнять в актуальных условиях поставщика.


| Сервис                              | Можно ли использовать | Условия использования                                                                                                                                                                     | Если нельзя — причина                                                                                                                                                                            |
| ----------------------------------- | --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **HeadHunter**                      | ✅ Да                  | Бесплатный API (с ограничениями по rate limit и данным)                                                                                                                                   | —                                                                                                                                                                                                |
| **Зарплата.ру**                     | ✅ Да                  | Отдельный API (`api.zp.ru`), экосистема HeadHunter; регистрация приложения, лимиты и объём данных по правилам площадки                                                                    | —                                                                                                                                                                                                |
| **SuperJob**                        | ✅ Да                  | Платный API + покупка тарифа (доступ к базе резюме)                                                                                                                                       | —                                                                                                                                                                                                |
| **Avito Работа**                    | ⚠️ Частично           | API только для работы с объявлениями и откликами                                                                                                                                          | ❌ Нет API для поиска/выгрузки кандидатов                                                                                                                                                         |
| **Яндекс Работа**                   | ⚠️ Частично           | Размещение вакансий через фид (формат YVL/XML по правилам Яндекса); поиск по вакансиям в интерфейсе                                                                                       | ❌ Нет публичного API массовой выгрузки базы резюме; сервис в основном агрегирует вакансии                                                                                                        |

| **Jooble**                          | ❌ Нет (для РФ)        | —                                                                                                                                                                                         | ❌ Для работы с аудиторией в РФ непригоден: доступ из страны ограничен/нестабилен (в т.ч. из‑за блокировок)                                                                                       |
| **Rabota (Mail.ru, экосистема VK)** | ❌ Нет                 | Поиск вакансий и резюме в интерфейсе портала                                                                                                                                              | ❌ Публичного API для интеграции по базе кандидатов не публикуется                                                                                                                                |
